feat: 配置 Appium 运行环境和依赖检测工具

为大麦票务助手项目添加完整的 Appium 环境检测和自动化安装工具，解决"未检测到 Appium 运行环境"的问题。

## 🎯 主要改进

### 1. 环境检测脚本 (`scripts/check_appium_env.py`)
- ✅ 自动检测 Node.js、npm、Appium Server 安装状态
- ✅ 检测 Python 包依赖（Appium-Python-Client、selenium、pydantic）
- ✅ 检测 ADB 工具和已连接设备
- ✅ 检测 Android SDK 环境变量配置
- ✅ 检测 Appium 驱动（UIAutomator2）
- ✅ 提供详细的修复建议和安装指导
- ✅ 彩色输出，清晰易读
- ✅ 跨平台支持（Windows/Linux/macOS）

### 2. 自动安装脚本
#### Windows 版本 (`scripts/install_appium.bat`)
- ✅ 自动安装 Appium Server 和 Appium Doctor
- ✅ 自动安装 UIAutomator2 驱动
- ✅ 安装 Python 依赖包
- ✅ 提供 ADB 和环境变量配置指导
- ✅ 自动运行环境检测验证

#### Linux/macOS 版本 (`scripts/install_appium.sh`)
- ✅ 自动检测操作系统和发行版
- ✅ 自动安装 Node.js、npm（通过包管理器）
- ✅ 自动安装 Appium Server 和相关工具
- ✅ 安装 Python 依赖包
- ✅ 提供 ADB 安装指导（macOS 支持 Homebrew）
- ✅ 环境变量配置建议

### 3. 配置文件和文档
- ✅ 完整的 Appium 配置模板 (`config/appium_config.example.json`)

## 🎓 使用方式

### 快速开始
```bash
# 1. 检查环境
python scripts/check_appium_env.py

# 2. 自动安装
scripts\install_appium.bat      # Windows
./scripts/install_appium.sh     # Linux/macOS

# 3. 验证安装
python scripts/check_appium_env.py
```

### 故障排查
```bash
# 运行环境检测获取详细信息
python scripts/check_appium_env.py

# 使用 Appium Doctor 诊断
appium-doctor --android
```
